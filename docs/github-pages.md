# Publicar no GitHub Pages

## Caminho simples

1. Crie um repositório no GitHub chamado `finance-core`.
2. Envie os arquivos deste projeto para a branch `main`.
3. Abra `Settings`.
4. Entre em `Pages`.
5. Em `Build and deployment`, selecione:
   - Source: `Deploy from a branch`;
   - Branch: `main`;
   - Folder: `/root`.
6. Salve.

Depois de alguns minutos, o GitHub mostra o link publicado.

## Instalar no iPhone

Abra o link publicado pelo Safari e toque em:

```text
Compartilhar > Adicionar à Tela de Início
```

O app abre como `Finance Core`.

## Arquivos importantes

- `index.html`
- `styles.css`
- `app.js`
- `manifest.webmanifest`
- `sw.js`
- `.nojekyll`
